# CineLog - Proyecto Bases de Datos 1

CineLog es una aplicación web académica desarrollada para la asignatura Bases de Datos 1.  
La aplicación permite consultar un catálogo de películas y series, iniciar sesión, registrar usuarios, calificar contenidos, crear listas personalizadas y escribir finales alternativos.

## Tecnologías utilizadas

- Java Spring Boot
- Angular
- MySQL
- Hibernate / JPA
- JWT
- HTML, SCSS y TypeScript
